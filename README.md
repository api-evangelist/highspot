# Highspot

Highspot is a sales enablement platform offering a REST API for managing content, training programs, pitch analytics, CRM integrations, and buyer engagement tracking. The API provides access to spots, items, users, groups, pitches, and analytics, with API key authentication using client key and secret credentials. Highspot also offers an MCP Server enabling LLMs to search content, manage pitches, generate Digital Rooms, and access Insights Layer APIs for AI-driven sales workflows.

## APIs

- **Highspot API** — REST API for content, users, groups, pitches, and analytics
- **Highspot MCP Server** — Model Context Protocol server for AI agent integration

## Authentication

API key authentication using a Key + Secret pair obtained from Highspot account settings under Developer > Basics. OAuth 2.0 is also supported for user-delegated authorization flows.

## Plans

Three tiers with custom pricing (contact sales for quotes):

- **Good — Equip and Engage**: Content management, AI search, CRM integration
- **Better — Train and Practice**: Adds skills management, adaptive learning
- **Best — Coach and Reinforce**: Full platform including MCP Server and Insights Layer APIs

Median annual contract value: ~$60,005 (range: $21,066–$178,451).

## Rate Limits

- MCP Server: 100 API calls per connection per 60 seconds
- REST API: Plan-dependent, not publicly documented
